## 🌟 Project Mission
As a **Student Digital Ambassador**, my role is to empower my peers to navigate the digital world with confidence, safety, and professionalism. This portfolio is a curated collection of resources designed to improve digital literacy across the campus—from building a professional online identity to identifying sophisticated cyber threats.

## 📂 Project Modules & Deliverables

### 🎨 Task 1: Digital Literacy Awareness
* **Focus:** Understanding the core pillars of digital literacy.
* **Deliverable:** A high-impact infographic created via Canva covering safe internet practices and essential student tools.
* **Link:** [Explore Task 1](./task-1-presentation/)

### 🚀 Task 2: Building a Professional Portfolio
* **Focus:** Establishing a foundational digital presence.
* **Deliverable:** Optimized profiles on LinkedIn, GitHub, and Kaggle, including a long-term "Tech-Art" career utilization plan.
* **Link:** [Explore Task 2](./task-2-portfolio/)

### 💻 Task 3: Coding & Cloud Collaboration
* **Focus:** Demonstrating technical agility and coordination.
* **Deliverable:** Completion of HackerRank Python challenges and the development of a Google Forms-based awareness quiz.
* **Link:** [Explore Task 3](./task-3-platforms/)

### 📧 Task 4: Professional Communication
* **Focus:** Mastering digital etiquette and responsible social media use.
* **Deliverable:** Formal email drafts for academic/corporate scenarios and a 10-point Social Media "Do's & Don'ts" guide.
* **Link:** [Explore Task 4](./task-4-email-etiquette/)

### 🛡️ Task 5: Cybercrime Prevention
* **Focus:** Recognizing and reporting online financial threats.
* **Deliverable:** A detailed case study on UPI/QR code fraud and a comprehensive safety checklist for Indian students.
* **Link:** [Explore Task 5](./task-5-cybercrime/)
